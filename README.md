# Good Feels Action: Stop the Hemp Ban Landing Page

A Vite + React + TypeScript one-page advocacy landing page styled in a Good Feels-inspired visual system.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important edits

Replace the placeholder petition URL in:

```txt
src/data/siteContent.ts
```

Search for:

```txt
#petition-link-goes-here
```

Before public launch, confirm any legal/compliance disclaimer language needed for civic advocacy.
